Projects

DeepRF

Per-pixel features from a pretrined deep convolutional neural network plus Random Forest. Shown to be useful for many tasks.

Code: Python, Caffe, Scikit-learn

Collaborator: Erik Rodner

Bird Part Localization (Fine-grained recognition)

This is basically an application of DeepRF. A fast way to detect bird parts (e.g. head) using deep features and random forests. Then using the detected part locations we can do fine-grained recognition on CUB-200-2011. Our mean accuracy is 74.47% which is comparable with state-of-the-art (76.3%) while being much faster.

Code: Python, Caffe, Scikit-learn

Collaborator: Erik Rodner

Supervised Detection of Field Lines

An annotator in flask and javascript. Plus the edge detection code of Piotr Dollar.
